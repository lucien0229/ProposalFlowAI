# 任务02-管理端总览

## 1. 任务目标

交付 Admin Overview 页面与对应聚合查询，展示平台整体规模、近期增长、关键转化与重点提醒。

## 2. 参考文档

- `docs/admin/ProposalFlow AI｜运营管理端 PRD v1.0.md`
- `docs/admin/ProposalFlow AI｜运营管理端指标与口径说明 v1.0.md`
- `docs/admin/ProposalFlow AI｜运营管理端 API 需求说明 v1.0.md`

## 3. 前置条件

- Admin shell 已可用
- `activity_logs` 与基础聚合 query 能力已具备

## 4. 主要工作内容

- 实现 `GET /api/v1/admin/overview`
- 返回平台级总览卡片所需数据
- 实现 Overview 页面：核心规模指标、近期增长、核心转化、重点提醒区
- 接入 7d / 30d 时间窗口切换与下钻跳转

## 5. 交付物

- Overview API
- Overview 页面
- 对应指标口径测试

## 6. 验收标准

- 指标命名体现对象与时间窗口
- Overview 数值与口径文档一致
- 指标卡和提醒项可跳转到过滤后的列表

## 7. 风险与边界

- 不让前端拼装多个资源接口来凑卡片
- 不把 user-level 指标混入 workspace-level 转化分母

## 8. 与上下游任务关系

- Workspaces / Subscriptions 页面可承接 Overview 下钻过滤
