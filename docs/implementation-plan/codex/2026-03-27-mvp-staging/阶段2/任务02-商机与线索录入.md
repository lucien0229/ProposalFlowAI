# 任务02-商机与线索录入

## 1. 任务目标

交付 Opportunities List 与 Opportunity Overview / Lead Intake，使用户能创建机会、录入原始输入、上传 PDF，并触发 Lead Brief 生成前的最小准备。

## 2. 参考文档

- `docs/web/ProposalFlow AI｜页面清单与页面级需求 v1.0.md`
- `docs/web/ProposalFlow AI｜API 设计 v1.0.md`
- `docs/share/ProposalFlow AI｜数据库设计 v1.0.md`

## 3. 前置条件

- Dashboard 与 route guard 已可用
- 文件处理基础链路已建立

## 4. 主要工作内容

- 实现 Opportunities List（搜索、状态过滤、archive/unarchive）
- 实现 Opportunity Overview / Lead Intake 页面
- 接入 raw input 保存、PDF upload-url / complete / detail / retry
- 展示 progress stepper、基础字段、文件状态与失败重试

## 5. 交付物

- Opportunities List 页面
- Opportunity Overview / Lead Intake 页面
- 对应 product API 与前端状态管理

## 6. 验收标准

- 用户可以创建 Opportunity 并保存基础字段
- 可粘贴 raw input、上传 PDF，并看到 `uploaded/processing/ready/failed`
- PDF 失败时可 retry 或改手动输入
- 满足最小条件后可进入 Lead Brief 流程

## 7. 风险与边界

- 不在本 task 内实现 Lead Brief 页面本身
- 文件处理只要求文本抽取，不做复杂版面解析
